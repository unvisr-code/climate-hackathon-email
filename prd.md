# 경기 기후 바이브코딩 해커톤 단일 HTML 이메일 PRD

## 1. 산출물 개요

* **파일명(예시)**: `hackathon_notice.html`
* **형태**: 단일 HTML 파일 1개
* **사용 방식**:

  1. 로컬에서 브라우저(Chrome 등)로 `hackathon_notice.html`을 연다.
  2. 화면에서 `Ctrl+A` → `Ctrl+C`로 전체 복사한다.
  3. Gmail 새 메일 작성 창에서 본문 영역을 클릭 후 `Ctrl+V`로 붙여넣는다.
  4. 제목 입력 후, 수신자/Bcc 추가하여 발송한다.

## 2. 이메일 목적

1. **장소 변경 안내**

   * 행사 일시는 동일하지만, 행사 장소가 변경되었음을 명확히 안내한다.
   * 새 장소: `경기도 수원시 영통구 도청로 52, 3층 강당 (경기신용보증재단 강당)`
   * 기존 행사 안내 페이지(예: 기존 장소 정보)는 **이메일에서 별도로 언급하거나 링크로 전달하지 않는다.**
   * 기존에 인지하고 있던 장소와 완전히 다른 곳이 아니라는 점(동일 도청 인근/수원 권역 내 이동임)을 간단히 언급해 심리적 부담을 낮춘다.

2. **Pre-Academy 가이드 안내 (CTA)**

   * 행사 전 개발 환경 세팅/Claude Code 설치 등 준비를 위한 Pre-Academy 페이지를 안내한다.
   * 이미 Pre-Academy 셋업을 완료한 사람은 **추가 진행이 필요 없다는 문구**를 포함한다.
   * Pre-Academy URL: `https://climate-gg-pre.vercel.app/`

3. **네이버 지도 링크 제공**

   * 참가자가 정확한 위치를 쉽게 확인할 수 있도록 **네이버 지도 링크를 버튼/텍스트로 제공**한다.
   * 네이버지도 URL:

## 4. 디자인 / 레이아웃 요구사항

### 4.1. 전체 레이아웃

* **폭**: 600px 고정 카드 레이아웃
* **정렬**: 화면 중앙 정렬
* **바깥 배경색**: `#f6f7f8` (연한 회색톤)
* **카드 배경색**: `#ffffff` (흰색)
* 구조:

  * 최상위 `<table width="100%">` 안에 중앙 정렬된 `<table width="600">` 사용
  * 큰 틀:

    1. 헤더 영역 (파란색 배경, 행사명)
    2. 본문 영역 (안내 텍스트 + 버튼들)
    3. 푸터 영역 (저작권 및 안내 문구)

### 4.2. 헤더 디자인

* `<td>`에 직접 inline style 지정
* 스타일 요구사항:

  * `background-color: #003d99;` (진한 블루 계열)
  * `padding: 24px;`
  * `text-align: center;`
  * `color: #ffffff;`
  * `font-size: 24px;`
  * `font-weight: bold;`
* 내용 예시:

  * "경기 기후 바이브코딩 해커톤" + 이모지(🛰️, 🌏 등) 1개 정도

### 4.3. 본문 디자인

* 패딩: `padding: 24px;`
* 폰트:

  * `font-family: Arial, sans-serif;`
  * `font-size: 14px ~ 15px;`
  * `line-height: 1.6;`
  * `color: #333333;`
* 주요 문장(장소, 일시, Pre-Academy 관련 키워드)은 `<b>` 태그로 볼드 처리.
* 이모지(`📅`, `📍`, `📘`, `🎯` 등)를 적절히 사용해 정보 덩어리를 시각적으로 구분.

### 4.4. 버튼(CTA) 스타일

1. **네이버 지도 링크 버튼 (선택적으로 적용)**

   * 텍스트: 예) "네이버 지도에서 위치 확인하기"
   * 스타일 예시:

     ```html
     <a href="{네이버지도URL}"
        style="display:block; width:100%; text-align:center; margin:12px 0; padding:12px 0; background:#00c73c; color:#ffffff; text-decoration:none; border-radius:6px; font-weight:bold; font-size:14px;">
       네이버 지도에서 위치 확인하기
     </a>
     ```
   * 색상: 네이버를 연상시키는 초록 (`#00c73c`) 사용 가능 (선택 사항).

2. **Pre-Academy 가이드 버튼 (주요 CTA)**

   * 텍스트: "Pre-Academy 가이드 확인하기"
   * 스타일 예시:

     ```html
     <a href="https://climate-gg-pre.vercel.app/"
        style="display:block; width:100%; text-align:center; margin:16px 0; padding:14px 0; background:#0073e6; color:#ffffff; text-decoration:none; border-radius:6px; font-weight:bold; font-size:15px;">
       Pre-Academy 가이드 확인하기
     </a>
     ```
   * 색상: 파란색 계열 `#0073e6`

### 4.5. 푸터 디자인

* 위치: 카드 맨 아래 `<tr>`
* 스타일:

  * `background-color: #eaeaea;`
  * `padding: 12px 16px;`
  * `font-size: 11px;`
  * `color: #777777;`
  * `text-align: center;`

## 5. HTML 기술 요구사항

1. **테이블 기반 레이아웃**

   * 이메일 클라이언트 호환성 확보를 위해 `<div>` 기반 flex/grid 레이아웃이 아닌 `<table>` 위주 구성.

2. **inline style 사용**

   * `<style>` 태그, 외부 CSS 파일은 사용하지 않고 각 요소에 `style="..."`를 직접 지정.

3. **스크립트 미사용**

   * `<script>` 태그 사용 금지.

4. **이미지 미사용**

   * 본 PRD에서는 로고/배너 이미지 없이 순수 텍스트 + 색상만 활용.
   * (추후 버전에서 `<img>` 사용을 고려할 수 있으나 현재 범위에서는 제외)

5. **문자 인코딩 및 기본 HTML 구조**

   * 상단에 다음 메타 태그 포함:

     ```html
     <!DOCTYPE html>
     <html lang="ko">
     <head>
       <meta charset="UTF-8" />
       <title>경기 기후 바이브코딩 해커톤 안내</title>
     </head>
     <body>...
     ```
